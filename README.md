## 📁 文件目录与功能说明


| 文件 / 文件夹 | 类型 | 说明 |
| :--- | :--- | :--- |
| `js/` | 📁 文件夹 | 存放 JavaScript 核心脚本（如 12306 等相关脚本） |
| `tubiao/` | 📁 文件夹 | 存放图标资源文件 |
| `12306.plugin` | ⚙️ 插件 | 12306 相关功能插件 |
| `ck.plugin` | ⚙️ 插件 | Cookie 获取或管理插件 |
| `daili.plugin` | ⚙️ 插件 | 代理相关功能插件 |
| `myxj.plugin` | ⚙️ 插件 | 名医汇（或美颜相机等）相关功能插件 |
| `wannianli.plugin` | ⚙️ 插件 | 万年历功能插件 |
| `wpsvip.plugin` | ⚙️ 插件 | WPS VIP 相关功能插件 |
| `Adblock-Plus.txt` | 📝 规则 | Adblock Plus 广告拦截规则文本 |
| `Apingbi.list` | 📝 分流 | A 站（或特定应用）屏蔽与分流列表 |
| `Azhilian.list` | 📝 分流 | 直连域名/IP 列表（不走代理） |
| `Telegram.list` | 📝 分流 | Telegram 官方节点分流列表 |
| `TelegramNL.list` | 📝 分流 | Telegram 荷兰（NL）节点专用分流列表 |
| `TelegramSG.list` | 📝 分流 | Telegram 新加坡（SG）节点专用分流列表 |
| `TelegramUS.list` | 📝 分流 | Telegram 美国（US）节点专用分流列表 |
| `Atubiao.md` | ℹ️ 文档 | 图表相关的说明文档 |
| `README.md` | ℹ️ 文档 | 项目主页说明文件（即本文件） |
